Engine

Rdzen terminalowego silnika 2D z modulami.

Powrot do spisu

Rola

`engine` dostarcza petle gry, systemy i API, ktore wykorzystuja mody.

Stack technologiczny

Rust + `ratatui` + `crossterm` jako fundament terminalowego silnika 2D.

Integracja z modem

Engine przyjmuje zrodlo moda jako folder lub plik `.zip`.

Kazde zrodlo musi zawierac plik wejscia `mod.yaml`.

`mod.yaml` musi zawierac pole `entrypoint` wskazujace scene (np. `/scenes/intro.yml`).

Glowna struktura to `ShellEngine` (rdzen uruchamiania i walidacji moda).
