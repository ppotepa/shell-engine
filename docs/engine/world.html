World (Component Registry)

Home / Engine / World

Overview

World is a type-map component registry. It stores arbitrary types keyed by their TypeId. Components are stored in one of two maps:

MapLifetimeTypical contents
singletonsEntire runEventQueue, Buffer, TerminalRenderer, SceneLoader
scopedOne sceneScene, Animator

get and get_mut search singletons first, then scoped. Each type must be unique across both maps.

Methods

register<T>(&mut self, val: T) -> &mut Self

pub fn register<T: Any + Send + Sync + 'static>(&mut self, val: T) -> &mut Self

Inserts a singleton component. Replaces any existing value of the same type.

world.register(EventQueue::new());
world.register(Buffer::new(term_w, term_h));

register_scoped<T>(&mut self, val: T) -> &mut Self

pub fn register_scoped<T: Any + Send + Sync + 'static>(&mut self, val: T) -> &mut Self

Inserts a scoped component. Cleared on every scene transition via clear_scoped().

world.register_scoped(scene);
world.register_scoped(Animator::new());

get<T>(&self) -> Option<&T>

pub fn get<T: Any + 'static>(&self) -> Option<&T>

Returns a shared reference to the component of type T, or None if not registered.

if let Some(scene) = world.get::<Scene>() {
    println!("active scene: {}", scene.id);
}

get_mut<T>(&mut self) -> Option<&mut T>

pub fn get_mut<T: Any + 'static>(&mut self) -> Option<&mut T>

Returns an exclusive mutable reference to the component of type T.

Calling get_mut requires a mutable borrow of World. You cannot hold any other borrow of World (including a previous get result) at the same time. Snapshot data from immutable borrows before calling get_mut.

// Correct pattern:
let scene_bg = world.get::<Scene>().map(|s| s.bg_colour.clone()); // dropped
if let Some(buf) = world.get_mut::<Buffer>() {
    buf.fill(scene_bg.unwrap_or_default());
}

clear_scoped(&mut self)

pub fn clear_scoped(&mut self)

Drops all scoped components. Called on every scene transition to release the current Scene and Animator before registering the next ones.

world.clear_scoped();
world.register_scoped(new_scene);
world.register_scoped(Animator::new());
