Mod: Shell Quest

Modyfikacja dzialajaca na silniku, rozwijana w `mod/shell-quest`.

Powrot do spisu

Rola

Mod definiuje zawartosc i logike gry specyficzna dla Shell Quest.

Punkt podlaczenia do engine

Shell Quest jest ladowany przez app i przekazywany do engine jako pakiet moda.

Plik `mod.yaml` jest entrypointem konfiguracji i musi wskazywac `entrypoint: /scenes/intro.yml` (lub inna scene).

Kontrakty YAML

Zmiany w plikach YAML wymagaja utrzymania schematow w `schemas/`:

  • `schemas/mod.schema.yaml`
  • `schemas/scene.schema.yaml`
