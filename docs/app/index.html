Overview

The app crate is the entry point binary. It creates a ShellEngine instance, points it at the mod source directory, and calls run() to hand control to the engine.

Flow

  1. Application starts.
  2. ShellEngine is constructed with the relative path mods/shell-quest/ (or SHELL_QUEST_MOD_SOURCE override).
  3. The engine validates mod.yaml against the mod schema.
  4. engine.run() enters the game loop and does not return until the user quits.

Source

// app/src/main.rs
fn main() {
    let engine = ShellEngine::new("mods/shell-quest/");
    engine.run();
}
