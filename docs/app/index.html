App

Warstwa startowa uruchamiajaca engine z wybranym modem.

Powrot do spisu

Rola

`app` odpowiada za punkt wejscia aplikacji i przekazanie kontroli do `engine`.

Przeplyw (aktualny)

  1. Start aplikacji.
  2. Utworzenie `ShellEngine` z relatywna sciezka `mod/shell-quest/`.
  3. Walidacja pliku `mod.yaml` przez engine.
  4. Uruchomienie engine z kontekstem moda.
