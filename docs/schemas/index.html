Overview

YAML contracts for mod and scene configuration files, defined as JSON Schema 2020-12. Schema files live in schemas/ at the repository root.

Schema files

  • schemas/mod.schema.yaml — validates mod.yaml manifests
  • schemas/scene.schema.yaml — validates scene *.yml files

Documentation

  • Mod Schema Reference — mod.yaml fields (entrypoint, terminal requirements, FPS) + virtual buffer design draft
  • Scene Schema Reference — every field documented with type, default, and description
  • Effects Catalogue — all built-in effects with parameters and YAML examples
