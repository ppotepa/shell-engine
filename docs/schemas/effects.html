Built-in Effects

Home / Schemas / Built-in Effects

Overview

Effects are named visual transformations applied to a buffer region over a duration. They are declared inside steps[].effects[] in scene or layer stages. All effects receive a progress value (0.0 to 1.0) each frame, optionally transformed by an easing function.

Usage notes: scene — typically used on scene stages (full buffer); layer — typically used on layer stages (row region); loopable — safe to use with loop: true.

Easing functions

All effects accept an optional params.easing field that transforms the raw linear progress before it is passed to the effect implementation.

ValueFormulaBehaviour
linear (default)tConstant rate throughout.
ease-int^2Starts slow, accelerates.
ease-outt(2-t)Starts fast, decelerates.
ease-in-outcubic HermiteSlow start and end, fast middle.

Effects

crt-on

scene

Simulates a CRT monitor powering on. Content is revealed from the vertical centre row outward as progress increases. At progress 0.0 the entire buffer is blank; at 1.0 the full buffer is visible.

ParamTypeDefaultDescription
easingstringlinearRecommended: ease-out for natural feel.
- name: crt-on
  duration: 5000
  params:
    easing: ease-out

clear-to-colour

scene

Fills the region with a solid colour at progress 1.0. No intermediate blend — the fill snaps to the target colour when the step completes. Use after content effects to blank the buffer between scenes.

ParamTypeDefaultDescription
colourcolourblackTarget fill colour.
easingstringlinearEasing (affects when the threshold fires).
- name: clear-to-colour
  duration: 3000
  params:
    colour: black
    easing: linear

power-off

scene

Simulates a CRT powering off. Content collapses from the edges toward the centre horizontal line as progress increases. The inverse of crt-on.

ParamTypeDefaultDescription
easingstringlinearRecommended: ease-in for a snap-off feel.
- name: power-off
  duration: 1500
  params:
    easing: ease-in

fade-in

layer

Reveals a region row-by-row from the top as progress increases. At progress 0.0 the entire region is blank; at 1.0 all rows are visible.

ParamTypeDefaultDescription
easingstringlinearRecommended: ease-in for a gentle appear.
- name: fade-in
  duration: 1000
  params:
    easing: ease-in

fade-out

layer

Hides a region row-by-row from the top as progress increases. At progress 0.0 all rows are visible; at 1.0 the entire region is blank. The inverse of fade-in.

ParamTypeDefaultDescription
easingstringlinearEasing applied to the hide progress.
- name: fade-out
  duration: 500

scanlines

scene loopable

Applies an ambient CRT scanline effect: dims every other row in the region by setting its background to black. Progress is ignored — the effect is stateless and renders identically at every progress value. Designed to be used with loop: true.

ParamTypeDefaultDescription
No parameters. All params are ignored.
- name: scanlines
  duration: 2000
  loop: true

Adding a custom effect

Custom effects with unknown names are silently ignored (no-op). To add a new built-in effect, add a match arm to apply_effect() in engine/src/effects.rs and add a corresponding entry to the schema's effect.properties.name description.
