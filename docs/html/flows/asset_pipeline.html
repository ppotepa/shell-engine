Overview

Assets flow: mod directories or ZIP packages → asset loader (engine/src/asset_loader.rs) → image/OBJ/font loaders → rasterizer/compositor. Confirmed from code.

Stages (high-level)

  1. Load mod (mod_loader.rs)
  2. Discover assets via directory scan or ZIP manifest
  3. Load on-demand: images (with packing), OBJs (with loader), fonts (rasterize)
  4. Cache in asset cache
  5. Use in compositor during frame render
