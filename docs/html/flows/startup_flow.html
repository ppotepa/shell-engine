Initialization sequence (confirmed)

  1. Parse CLI args (launcher.sh or cargo run -p app)
  2. Resolve mod source (shell-quest by default, or --mod-source)
  3. Load mod (mod_loader.rs)
  4. Run startup checks (pipelines/startup/checks)
  5. Load initial scene (scene_loader.rs)
  6. Compile scene (engine-authoring)
  7. Materialize SceneRuntime
  8. Enter game_loop

Startup checks

Schema validation, effect registry check, font manifest check, asset existence check, scene graph validation.
