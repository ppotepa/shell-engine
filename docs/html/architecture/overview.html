System Layers

  • App Layer: CLI, mod selection, launcher
  • Engine Layer: Game loop, event queue, system scheduler, scene runtime
  • Core Layer: Scene model, sprites, layers, behaviors, effects, animations
  • Authoring Layer: YAML compilation, schema, normalization
  • Editor Layer: TUI editing, asset management, hot-reload

Key Components

  • World: Container for all active entities, systems state
  • SceneRuntime: Materialized scene at runtime, sprite/layer live state
  • EngineEvent: Event queue (KeyPressed, MouseMoved, Tick, Quit)
  • Systems: Animator, Compositor, Renderer, BehaviorSystem, SceneLifecycleManager

Startup Sequence

Launcher → select mod → load manifest → load/compile scenes → initialize engine → frame loop

Frame Pipeline

Event poll → Behavior execution → Animation update → Compositor layout → Renderer output → Display
