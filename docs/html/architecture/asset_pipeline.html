Asset Types

  • Images: PNG, indexed-color (1-4 bits), loaded as bitmap sprites
  • Fonts: TTF, rasterized on-demand to bitmap
  • Objects: OBJ geometry, converted to ASCII art

Load Flow

  1. Asset Reference: YAML scene specifies image_path, font_path, obj_path
  2. Mod Loader: Locate asset relative to mod root (directory or .zip)
  3. Format Detection: Determine asset type from extension
  4. Load Handler: Deserialize (image → bitmap, font → glyph cache, obj → mesh)
  5. Cache: Assets cached in World for scene re-use

Image Processing

  • Indexed PNG → 1/2/4-bit bitmap
  • Dithering optional (Floyd-Steinberg)
  • Character rendering modes: halfblock, braille, cell

Font Rasterization

TTF → fontdue → bitmap glyphs on-demand. Cached per font + size combo.
