Phases

  1. CLI Parse: app::main() → Cli::parse() → mod selection
  2. Mod Load: Load manifest.yaml from selected mod
  3. Scene Compilation: For each scene YAML, compile to Scene (engine-authoring)
  4. Engine Init: Create World, initialize all systems
  5. Frame Loop: Begin game_loop()

Manifest Loading

ModManifest describes scenes, assets, and global settings. Scene paths are resolved relative to mod root.

Scene Compilation

YAML → SceneDocument → normalize → compile with filters → Scene (runtime)

System Initialization

Animator, Compositor, Renderer, BehaviorSystem, AudioSystem, MenuSystem all initialized from config.
