Responsibilities

The engine crate implements the runtime: event model, fixed-timestep game loop, scene lifecycle manager, systems (animator, behavior, audio, compositor, renderer), rasterizer, and asset loading.

Key files (evidence)

  • engine/src/game_loop.rs — loop, event poll, MouseCaptureGuard
  • engine/src/scene_runtime.rs — SceneRuntime, TerminalShellState
  • engine/src/systems/* — animator/compositor/renderer
