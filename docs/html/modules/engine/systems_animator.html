Role

The animator system advances authored animation parameters and emits transition events when stages/steps complete. It is the first system executed in the per-frame systems order.

Responsibilities (confirmed)

  • Advance animations per tick
  • Emit transitions that SceneLifecycleManager consumes
  • May produce post-animator events that must be drained in the same frame

Source evidence

engine/src/systems/animator.rs and game_loop.rs (post-animator drain)
