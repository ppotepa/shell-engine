Overview

Rasterizer components convert font glyphs and images into terminal-friendly bitmaps. Key files: engine/src/rasterizer/*.rs, and tools/ttf-rasterizer for raster generation.

Notes

Rasterizer supports multiple backends and produces glyph atlases used by the renderer. Exact format and packing inferred from rasterizer code (see files).
