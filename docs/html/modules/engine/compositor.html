Overview

The compositor assembles layers and sprites into the virtual buffer before it is handed to the renderer. It handles layout, layering order, and effect application.

Confirmed responsibilities

  • Compose layers into the scene buffer (engine/src/systems/compositor/*)
  • Apply built-in effects to sprites (engine-core effects)
  • Coordinate with renderer for packing modes (cell/halfblock/quadblock/braille)

Evidence

engine/src/systems/compositor/* and engine-core/src/effects
