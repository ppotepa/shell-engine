Purpose

The renderer transforms the composed virtual buffer into terminal output using packing modes. Supported modes (from schema) include cell, halfblock, quadblock, and braille. Current runtime primarily uses halfblock (confirmed).

Implementation notes

  • Renderer systems located under engine/src/systems/renderer.rs and related render modules.
  • Packing mode is taken from scene.schema.yaml rendered-mode and can affect virtual resolution.
