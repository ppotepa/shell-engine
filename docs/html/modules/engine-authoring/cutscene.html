Overview

engine-authoring/src/compile/cutscene.rs compiles authored cutscene refs into timed image sprites. The compiler supports plug-in filters (CutsceneCompileFilter) for transformations such as duration scaling.

Responsibilities

  • Validate cutscene frame sequences (numeric chronology, no gaps)
  • Produce timed sprite sequences with appear_at_ms/disappear_at_ms
  • Expose filters for compile-time transformations
