Overview

engine-core/src/scene/model.rs defines the authored scene model types: Scene, Sprite, Layer, BehaviorSpec, and UI/theme types. These are the canonical data model used by engine-authoring and the engine runtime.

Key types

  • Scene — root authored declaration (id, title, stages, layers, objects)
  • Sprite — renderable pieces (bitmap, text, image)
  • Layer — compositing layer container
  • BehaviorSpec — declarative behavior binding for objects

Schema

Scene schema is defined at schemas/scene.schema.yaml which maps to these types. The schema includes scene_input profiles including terminal-shell.
