Overview

Cutscene maker is an editor panel for authoring stop-action sequences. It validates frame folder numeric chronology, allows per-frame delays and palette profiles, and saves cutscene YAML under mods/*/cutscenes/.

Features (confirmed)

  • Frame folder validation
  • Per-frame delay editing (bulk multiply)
  • Palette profile selection (faithful/retro-128/etc.)
  • Live preview
  • YAML persistence
