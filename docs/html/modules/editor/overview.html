Responsibilities

The editor is a TUI application (ratatui) for scene/effect/asset authoring. Key components: UI components for scene previews, cutscene preview, sidebar explorer, status bar, and state management. It consumes engine-core models and uses engine-authoring to compile scenes for preview.
