Overview

Shared authoring pipeline for Shell Quest YAML assets. This crate is the future source of truth for authored scene/object input: parsing, package assembly, normalization, validation, and schema metadata. Runtime execution and rendering stay in `engine`; shared runtime data stays in `engine-core`.
