Module engine-audio::lib

Canonical file-backed API module

Home / API Reference / Modules / engine-audio::lib

Overview

Audio backend abstraction and Rodio-based audio playback for the engine. Provides: - `AudioBackend` trait for pluggable audio implementations - `RodioAudioBackend` in-process WAV/MP3 playback - `NullAudioBackend` for tests and headless runs - `AudioProvider` trait for integration with any container (engine's World, etc) - `audio_system()` to flush commands each frame
