Module engine-audio::audio

Overview

Audio command queue and backend abstraction used by the engine's audio system. When audio is enabled, the engine uses an embedded [`RodioAudioBackend`] that plays WAV/MP3 files directly via the system audio device — no external process needed.

Source: engine-audio/src/audio.rs

Module docs

Audio command queue and backend abstraction used by the engine's audio system.

When audio is enabled, the engine uses an embedded [`RodioAudioBackend`] that plays WAV/MP3 files directly via the system audio device — no external process needed.
