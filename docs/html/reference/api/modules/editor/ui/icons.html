Overview

Icon glyph helpers with automatic fallback when Nerd Fonts are unavailable. The default output is ASCII-safe so icons still render on basic terminals. Set `SHELL_QUEST_ICON_THEME` to `emoji` or `nerd` to opt into richer glyphs.

Source: editor/src/ui/icons.rs

Module docs

Icon glyph helpers with automatic fallback when Nerd Fonts are unavailable.

The default output is ASCII-safe so icons still render on basic terminals. Set `SHELL_QUEST_ICON_THEME` to `emoji` or `nerd` to opt into richer glyphs.
