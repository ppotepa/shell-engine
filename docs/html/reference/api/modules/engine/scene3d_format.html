Overview

Parser for `.scene3d.yml` — describes a complete 3D scene with objects, lights, materials, camera, and named frames (static or animated clips). The parsed [`Scene3DDefinition`] is consumed by [`Scene3DPrerenderStep`] to pre-render every named frame into the [`Scene3DAtlas`] before a scene loads.
