Overview

Unified frame-skip oracle — coordinates all frame-skip decisions to prevent desynchronization. **Problem:** PostFX had its own frame-skip cache (every N frames), and Presenter had its own hash-based skip logic. When they disagreed, frames got duplicated or dropped, causing flickering and animation artifacts. **Solution:** Single oracle that both systems consult. Either all skip together, or all render together. No desynchronization possible.
