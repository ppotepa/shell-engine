Overview

Synchronous pre-render pass: renders each eligible OBJ sprite in the scene exactly once at its YAML-declared initial pose, storing the result in `ObjPrerenderedFrames`. Called during scene transition when `scene.prerender == true`. A sprite is eligible when `rotate_y_deg_per_sec` is zero or absent — i.e. it has a static rest pose. Dynamic lights (snap_hz, flicker_hz) are fine: we capture the initial frame exactly. The caller opts specific sprites out with `prerender: false` on the sprite itself. All eligible sprites are rendered in parallel via rayon.
