Overview

Streaming asset warmup — pre-loads OBJ meshes into the cache before rendering. Loads all OBJ sprites (static and dynamic) in parallel during scene preparation, so the compositor finds them already cached on first render instead of loading synchronously mid-frame. This eliminates the "first frame stutter" caused by on-demand mesh parsing.
