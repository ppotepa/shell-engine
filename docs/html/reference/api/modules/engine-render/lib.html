Overview

Abstract render backend trait enabling pluggable rendering (Terminal, OpenGL, D3D, Vulkan, WebGL). Core types: - `RenderBackend` trait: present frames, query capabilities, shutdown - `DisplaySink` trait: queue and flush frames (may be async) - `RenderFrame`: per-frame data (buffer, virtual size, present mode) - `RenderCaps`: capabilities (resolution, color depth, FPS)
