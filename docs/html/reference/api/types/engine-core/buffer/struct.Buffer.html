Overview

Double-buffered pixel bitmap for the terminal screen. `back` is written to by the compositor each frame. `front` mirrors what is currently visible on the terminal. Only cells that differ between back and front are flushed on each render. After flushing, `swap()` copies back → front.

Source evidence

Double-buffered pixel bitmap for the terminal screen.

`back` is written to by the compositor each frame. `front` mirrors what is currently visible on the terminal. Only cells that differ between back and front are flushed on each render. After flushing, `swap()` copies back → front.
