Overview

Scene preparation step: prerenders eligible `type: obj` sprites to in-memory canvases before the scene is activated. Runs only when `scene.prerender == true`. Individual sprites opt out with `prerender: false` on the sprite itself. Registers scoped [`ObjPrerenderedFrames`] and [`ObjPrerenderStatus::Ready`] into the world.
