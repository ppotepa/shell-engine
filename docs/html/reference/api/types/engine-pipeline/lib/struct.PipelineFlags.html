Source evidence

Feature flags for the render pipeline.

`PipelineFlags::default()` is always the safe, stable configuration. The `opt_*` fields are read once at startup to construct `PipelineStrategies`. Runtime systems use `PipelineStrategies`, not these booleans directly.

Expected: 1-5ms/frame unblocked on slow terminal emulators. Default: `false` (sync flush — no latency decoupling).
