Type summary

Confirmed from engine/src/scene_runtime.rs. TerminalShellState stores terminal input and output buffer for the interactive terminal-shell input profile.

Fields

  • controls: TerminalShellControls — authored controls (prompt_prefix, banner, commands)
  • input: tui_input::Input — text input buffer used to render prompt and edit text
  • output_lines: Vec — current output history (bounded by max_lines)
  • history: Vec, history_cursor: Option

Behaviour & built-ins

The runtime exposes built-in commands: clear, help, pwd, whoami, echo. The execute_command implementation is in scene_runtime.rs and pushes the command into the output_lines, updates history, and handles the built-ins. Custom commands may be provided by the scene via controls.commands.
