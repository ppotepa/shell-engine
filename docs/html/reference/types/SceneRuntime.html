Signature & role

SceneRuntime is the runtime representation of an authored Scene. It holds:

  • scene: Scene — authored model copy
  • maps for objects, layers, sprites
  • behaviors and resolver cache
  • optional terminal_shell_state: Option

Public responsibilities

  • Materialization and id resolution
  • Provide per-frame mutable state
  • Expose helper APIs for runtime systems (inferred from usage)

Dependencies

  • engine-core Scene model
  • TerminalShellState (optional)
