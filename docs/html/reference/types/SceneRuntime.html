SceneRuntime

struct — Materialized Scene Runtime State

Home / Types / SceneRuntime

Source

engine/src/scene_runtime.rs

Overview

SceneRuntime represents a loaded Scene with live state: sprite visibility, animation counters, behavior variables. One per active scene in World.

Key Fields

scene_id, sprites[], layers[], terminal_shell_state (Option), behavior_state

Lifecycle

Created on scene load, destroyed on scene unload
