Definition

EngineEvent is the central event enum used by the runtime. It carries input events (Key, Mouse), terminal resize, Tick, and Quit. Confirmed by inspection of engine/src/game_loop.rs and engine/src/events.rs.

Common variants (confirmed)

  • KeyPressed(KeyEvent)
  • MouseMoved { column, row }
  • TerminalResized { width, height }
  • Tick
  • Quit

Used by
