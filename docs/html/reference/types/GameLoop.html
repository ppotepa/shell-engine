Overview

game_loop.rs exposes game_loop(world: &mut World, target_fps: u16) which implements the fixed-timestep engine main loop. Responsibilities include input polling, event enqueueing, tick emission, event drainage and system ticks.

Important behaviour (confirmed)

  • MouseCaptureGuard RAII enables/disables mouse capture
  • is_quit_key helper treats Ctrl+C as quit
  • post-animator events are drained and processed in same frame
