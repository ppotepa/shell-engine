Overview

Schemas are JSON Schema YAML files that define scene, effect-param, and authoring constraints. Base schemas live at schemas/*.schema.yaml. Mod overlays are generated under mods/*/schemas/ via schema-gen.

Canonical files

  • schemas/scene.schema.yaml — defines Scene model, input profiles, layers, sprites
  • schemas/effect-params.schema.yaml — effect parameter constraints

Generation

schema-gen reads engine-authoring definitions and generates/validates mod overlays. Workflow: author YAML → cargo run -p schema-gen -- --all-mods → check diffs.
