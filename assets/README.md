# assets

Shared repository-level media assets and source files.

## Purpose

`assets/` holds loose media files that are useful during development and
documentation but are not organized as a runtime mod on their own.

## What lives here

- source audio project files such as `.aup3`
- exported audio clips such as `.wav`
- error/debug imagery used by tooling or docs

If an asset is intended for gameplay, it should usually live inside a specific
mod’s `assets/` directory instead.
