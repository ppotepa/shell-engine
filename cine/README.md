# cine

Cinematic and 3D source assets used for Shell Quest visual content.

## Purpose

`cine/` holds raw or intermediate visual assets used to build cinematic and 3D
content, including Blender files, exported GLB models, sculpt assets, and
shared source material.

## Typical contents

- Blender source files
- exported 3D models
- sculpt variants
- shared source images for intro/cinematic content

These are source assets for content production, not a standalone runtime module.
