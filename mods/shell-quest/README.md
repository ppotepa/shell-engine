# shell-quest

Primary playable content mod for Shell Quest.

## Purpose

This mod contains the real game content: scenes, objects, behaviors, assets,
schemas, and sidecar-backed terminal sequences used in the main player
experience.

## Contents

- `mod.yaml` — mod manifest and runtime defaults
- `scenes/` — authored scenes and scene packages
- `objects/` — shared object fragments
- `behaviors/` — mod-defined behaviors
- `assets/` — images, audio, fonts, and 3D assets
- `schemas/` — generated mod-local schema fragments
